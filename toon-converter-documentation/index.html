Percentage of tokens saved (typically 30-60%)

Cost Saved

Estimated API cost savings based on your configured token price

Token Estimation Method

The converter uses a heuristic-based estimation combining:

  • ‚Ä¢ Character count method: Approximately 4 characters per token
  • ‚Ä¢ Word count method: Words √ó 1.3 token multiplier
  • ‚Ä¢ Averaged result: Mean of both methods for accuracy

Accuracy Note: Token estimation is approximately 85-95% accurate. Different LLM providers use different tokenizers (OpenAI's tiktoken, Claude's tokenizer, etc.). However, the relative savings (JSON vs TOON) remain consistent across all tokenizers.

Troubleshooting

‚ùå "Invalid JSON" Error

Problem: You see an error message when pasting or typing JSON.

Common Causes:

  • ‚Ä¢ Trailing commas: {"name": "Alice",}
  • ‚Ä¢ Single quotes instead of double: {'name': 'Alice'}
  • ‚Ä¢ Comments in JSON: // This is not allowed
  • ‚Ä¢ Missing quotes around keys: {name: "Alice"}
  • ‚Ä¢ Unescaped special characters in strings

Solution: Use a JSON validator (like jsonlint.com) to fix your JSON before pasting.

‚ö†Ô∏è Auto-Convert Not Working

Problem: Nothing happens after pasting JSON.

Solution:

  • ‚Ä¢ Wait 500ms (half a second) for debounce delay
  • ‚Ä¢ Manually click the "Convert to TOON" button
  • ‚Ä¢ Check browser console for JavaScript errors (F12)
  • ‚Ä¢ Refresh the page if the tool is unresponsive

‚ÑπÔ∏è Token Savings Lower Than Expected

Problem: Showing only 10-20% savings instead of 30-60%.

Possible Reasons:

  • ‚Ä¢ Non-uniform data: Objects in arrays have different keys
  • ‚Ä¢ Deeply nested structure: 5+ levels of nesting
  • ‚Ä¢ Small dataset: Only 1-5 items in array
  • ‚Ä¢ Mixed types: Array contains both objects and primitives

Solution: TOON works best with uniform arrays of 10+ objects. Consider normalizing your data or sticking with JSON for this use case.

üóëÔ∏è History Not Saving

Problem: Conversions don't appear in history.

Causes & Solutions:

  • ‚Ä¢ Private/Incognito mode: localStorage doesn't persist. Use normal browsing mode.
  • ‚Ä¢ Browser storage disabled: Check browser settings to enable cookies/storage.
  • ‚Ä¢ Storage quota exceeded: Clear browser data or delete old history entries.
  • ‚Ä¢ Invalid conversion: History only saves successful conversions, not errors.

üìÅ Drag & Drop Not Working

Problem: File drop doesn't load JSON.

Checklist:

  • ‚Ä¢ ‚úì File has .json extension
  • ‚Ä¢ ‚úì File contains valid JSON
  • ‚Ä¢ ‚úì File size is reasonable (<10MB recommended)
  • ‚Ä¢ ‚úì Dragging directly over the input textarea area
  • ‚Ä¢ ‚úì Browser supports drag & drop (not IE11)

üêõ Still Having Issues?

If problems persist:

  1. 1. Clear browser cache: Ctrl+Shift+Delete (Windows) or Cmd+Shift+Delete (Mac)
  2. 2. Try a different browser: Chrome, Firefox, Safari, or Edge
  3. 3. Check browser console: Press F12 and look for error messages
  4. 4. Disable extensions: Ad blockers or privacy tools may interfere
  5. 5. Hard refresh: Ctrl+F5 (Windows) or Cmd+Shift+R (Mac)
